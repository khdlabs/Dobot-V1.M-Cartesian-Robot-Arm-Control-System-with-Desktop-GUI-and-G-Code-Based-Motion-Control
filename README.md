# Dobot-V1.M-Cartesian-Robot-Arm-Control-System-with-Desktop-GUI-and-G-Code-Based-Motion-Control
Dobot V1.M is an open-source Cartesian robot arm controlled via a desktop GUI using serial communication and G-code commands. Designed for education, prototyping, and automation, it supports precise motion control, real-time monitoring, and Python-based extensibility.
